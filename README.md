# 🏥 Doctor Appointment Booking System

A modern and responsive Doctor Appointment Booking Web Application built using React and Tailwind CSS.  
This project allows users to view doctor details, login, and simulate appointment booking with a payment interface.

---
## 📌 Features

- 👩‍⚕️ Doctor Profile Page
- 🔐 User Login / Authentication (Local Storage based)
- 📅 Appointment Booking Flow
- 💳 Payment Page (Demo Mode - Card & UPI UI)
- 🧭 Dynamic Navbar (Login / Logout)
- 📄 Footer with Routing
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite

---

## 🛠 Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Build Tool:** Vite
- **Version Control:** Git & GitHub
- **Deployment:** Vercel

## 🔄 Appointment Booking Flow

1. User visits homepage
2. Clicks "Make Appointment"
3. Views doctor details
4. Logs in
5. Proceeds to payment page
6. Payment success confirmation (Demo Mode)

---

## 💳 Payment Integration

This project includes a demo payment interface that simulates:

- Card Payments
- UPI / QR Payment

> ⚠️ Payment is in demo mode (no real transaction processing).
